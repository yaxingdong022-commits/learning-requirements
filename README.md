# 学习要求 & 学习计划

## 学习目标
**快速入行Web爬虫工程师**

---

## 教学要求
1. **客观评价** - 不能盲目鼓励，要指出问题
2. **按时实训** - 每隔几节课给实战练习
3. **高效率** - 教学要快、准、狠 以就业为目标
4. **综合实战** - 每次练习都融合多个已学知识点
5. **尊重自主** - 学生自行判断进度

---

## 课程进度
第6课：Selenium爬虫基础 - 数据提取
提取元素的文本、属性、链接

实战：爬取一个简单的商品列表
第7课：数据存储
保存到CSV文件
